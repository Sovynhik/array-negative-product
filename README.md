# ArrayNegativeProduct — Лабораторная работа по массивам

**Java-приложение** для вычисления **произведения элементов между двумя отрицательными числами** в одномерном массиве.

---

## Задача

> **Дан массив целых чисел.**  
> Найти **два первых отрицательных элемента** и вычислить **произведение элементов между ними**.  
> Если:
> - меньше двух отрицательных → вывести сообщение  
> - отрицательные рядом → вывести сообщение  
> - иначе → вывести произведение

---

## Пример работы

```text
Массив: [0, 2, -1, 4, 1, 3, 5, -11, 9]
→ Произведение между -1 и -11: 4 × 1 × 3 × 5 = 60
```

**Вывод:**
```
Исходный массив: [0, 2, -1, 4, 1, 3, 5, -11, 9]
Произведение элементов между двумя отрицательными: 60
```

## Как запустить

1. Убедитесь, что установлена **jdk1.8.0_144+**
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/array-negative-product.git
   ```
3. Перейдите в папку проекта и выполните:
   ```bash
   javac -d out -cp . src/ru/rsreu/savushkin0317/*.java src/ru/rsreu/savushkin0317/onedimarray/*.java src/ru/rsreu/savushkin0317/onedimarray/productfinder/*.java
   java -cp out ru.rsreu.savushkin0317.ClientRunner
   ```

## Особенности

- **Инкапсуляция**: `OneDimensionalIntArrayHandler` — безопасный доступ к массиву
- **Обработка ошибок**: `IndexOutOfBoundsException` с локализованным сообщением
- **Локализация**: все сообщения вынесены в `messages_ru_RU.properties`
- **Чистый код**: разделение ответственности, `StringBuilder`, `varargs`
